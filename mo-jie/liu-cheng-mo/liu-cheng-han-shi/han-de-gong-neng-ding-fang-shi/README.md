# 步驟函式的功能與設定

步驟函式大致上可分為幾大類：

1. **基本**
   1. 標籤【Label】
   2. 通知型對話框【Dialog】
   3. 互動型對話框【Button Dialog】
   4. 表達式【Expression】
   5. 延遲【Wait】
   6. 時間戳記【Time String】
2. **邏輯**
   1. 迴圈【While】
   2. 條件式【If】...【ElseIf】...【End】
3. **模組**
   1. 命令通訊連線【SmaCOM】
   2. 命令機器視覺【SmaVISION】
   3. 命令機械手臂【SmaROBOT】
   4. 命令運動控制【SmaMOTION】
   5. 命令數位訊號【SmaDIO】
4. **進階**
   1. 呼叫子流程【Subsequence】
   2. 觸發錯誤【Error】
   3. 流程控制【Action】

以上函式將會在接下來的段落中詳細說明
